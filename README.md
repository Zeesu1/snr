# 用法 Usage

Usage: snr [options] [command]

Options:
  -V, --version    output the version number
  -h, --help       display help for command

Commands:
  ls               查看镜像
  use              请选择镜像
  current          查看当前源
  test [registry]  测试镜像地址速度
  add              自定义镜像
  delete           删除自定义的源
  help [command]   display help for command
